# Security Policy

## Supported Versions

Neutron is currently in **beta**, and only the **latest release** is actively supported with security updates and patches.

| Version     | Supported          |
| ------------ | ------------------ |
| Latest Beta  | :white_check_mark: |
| Older Builds | :x:                |

As Neutron evolves rapidly during beta, older builds may contain unresolved security issues. Please always update to the most recent version for the latest fixes and improvements.

## Reporting a Vulnerability

If you discover a security vulnerability in Neutron, please report it directly and confidentially to **Yasakei**.

You can reach out via the official Neutron contact channel or email (if applicable). When reporting, include as much detail as possible about the issue, including:

- A clear description of the vulnerability  
- Steps to reproduce the issue (if possible)  
- The version or build you were using  

Once a report is received:
- You’ll receive an acknowledgment within a few days.  
- Verified issues will be prioritized and patched as soon as possible.  
- You’ll be notified once the issue is resolved or if further details are needed.

We appreciate your help in keeping **Neutron** secure for everyone.

— **Yasakei**
